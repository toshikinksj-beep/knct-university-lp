# KNCT UNIVERSITY — LP

SNSクリエイターの裏側で収益を作る「オペレーター」という働き方を学ぶ KNCT UNIVERSITY のランディングページ。Instagram などのSNSから **LINE 友だち追加** へ誘導することを目的としたモバイル特化の1枚構成。

## 特徴

- 📱 **モバイルファースト** — スマホでの見え方を最優先に設計
- 🎨 **Figma / Notion 風のミニマルデザイン** — 白黒基調＋要所のみ LINE グリーン
- 🟢 **LINE 誘導特化** — ナビ・ヒーロー・最終CTA・追従バーの全ボタンが LINE 追加へ直行
- ⚡ **依存ゼロの単一HTML** — ビルド不要、そのままホスティング可能

## 構成

1. ヒーロー（SNSで発信しなくても、収益化できる時代へ。）
2. こんな悩み、ありませんか？
3. オペレーターとは？（「発信する人」vs「売上を作る人」）
4. どうやって収益になるの？（報酬 20〜50%）
5. 実際の結果（受講生ケース）
6. なぜ結果が出るのか？
7. 向いていない人もいます
8. 無料特典の案内
9. 最終CTA

## セットアップ

公開前に、実際の LINE 友だち追加URLを設定してください。
`index.html` 末尾のスクリプト内：

```js
var LINE_URL = "https://lin.ee/your-line-id"; // ← 実際のURLに差し替え
```

全ての LINE ボタンが自動でこのURLを参照します。

## ローカルプレビュー

```bash
python3 -m http.server 4599
# → http://localhost:4599
```
